# kotlin-cli-gradle-seg-sieve-functional-prog

## Description
Find all primes less in a range. Follows functional programming practices.

## Tech stack
- kotlin

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
